What is Mono?

Mono is an open source, cross-platform implementation of C# and the CLR that is binary compatible with the .NET Framework. Mono has been in development since 2001 and is very mature.

Here's how it all started as told by Miguel de Icaza.

The Mono Project is sponsored by Xamarin, who now have a worldwide strategic partnership with Microsoft.

What is Sailfish?

Sailfish is a mobile GNU/Linux distribution with a user interface built on the Qt platform. Sailfish includes Qt5, QtQuick2 and Wayland.

The project is independent, open, partner-friendly, and created by a team with credible track record, now working at Jolla Ltd.

Jolla is set to create an ecosystem unlike others around Sailfish and welcomes everyone to join the voyage.

What is Mono for Sailfish?

Mono for Sailfish is a port of Mono to Sailfish, including an IDE integration for MonoDevelop. It is a continuation of the suspended MonoTizen Project.

How do I use Mono for Sailfish?

Development on Mono for Sailfish started in August 2014. There should be a usable MonoDevelop add-in and RPMs before the end of the year.

In the meantime, anybody is welcome to contribute to the project on Github. Any and all pull requests are welcome!
